# mobweb
一个快速制作移动宣传页面的工具

## 使用方法 ##
直接打开[http://75.75.242.55/](http://75.75.242.55/)使用，或搭建自己的服务

提前安装：[nodejs](http://nodejs.org)，[mongodb](http://www.mongodb.org/)，git
1. 下载代码

> git clone https://github.com/postor/html5-generator.git

2. 安装依赖

> cd mobweb

> npm install

3. 修改配置

> vi config.js

4. 启动服务

> node app.js

## 已有功能 ##
- 注册
- 登陆
- 创建页面
- 编辑页面、元素
- 元素进场效果
- 元素持续动画
